## 困境预设 + 解决方案

### 预设困境1：设备连接不上
| 可能的问题 | 预设解决方案 |
|----------|------------|
| 蓝牙断开 | 重新连接/检查驱动 |
| WiFi不稳定 | 用有线/靠近路由器 |
| 驱动版本太旧 | 更新驱动 |

**针对性的刻意练习**：检查连接流程练习

---

### 预设困境2：Demo打包失败
| 可能的问题 | 预设解决方案 |
|----------|------------|
| 证书问题 | 检查证书/重新申请 |
| 文件太大 | 压缩资源/减少面数 |
| 配置错误 | 对照教程检查 |

**针对性的刻意练习**：打包流程练习

---

### 预设困境3：性能卡顿
| 可能的问题 | 预设解决方案 |
|----------|------------|
| 面数太多 | 减面/烘焙 |
| 贴图太大 | 压缩贴图/合并纹理 |
| 实时渲染慢 | 切换到Eevee/降低精度 |

**针对性的刻意练习**：性能优化练习
