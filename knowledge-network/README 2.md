# 知识网络 Knowledge Network

> 英文版本：[README-en.md](README-en.md)

知识网络是 VIRTURA-SpacePort 的核心学习与实践体系，是一个**在线演进的智力装备**，为沉浸式体验、**数字内容创作**、**人工智能**领域的学习者和创作者提供从零基础到精通的完整路径。

---

## 什么是知识网络？

知识网络不是按阶段线性排列的，而是**10个相互关联的知识领域大类**，用户可以选择自己感兴趣的领域进行学习。

知识网络强调：
- **最短路径**：帮助零基础用户快速跨越鸿沟
- **多种实现方法**：每个模块列出2-3种方案
- **困境预设**：提前告知可能遇到的问题和解决方案
- **交叉学科**：知识领域之间相互关联
- **数字内容创作**：支持沉浸式体验、数字艺术创作
- **人工智能**：AI生成、AI辅助创作、本地模型部署

## 知识领域详解

目标：帮助零基础用户在30分钟内发布第一个Demo

- **Apple Vision Pro**
  - 最短路径：新手指南（30分钟内）

  - 官方资源筛选：2024-2025最新
  - 实战案例：180°视频→Blender处理→导出双目→App独立运行

- **Meta Quest**
  - 最短路径
