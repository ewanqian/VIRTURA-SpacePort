## 具身理论

### 什么是具身认知？

具身认知（Embodied Cognition）是指人类通过身体与环境的互动来形成认知的理论。

在VR/AR/XR中，具身理论帮助我们理解：
- 用户如何在虚拟空间中获得"在场感"
- 身体动作如何影响感知体验
- 空间交互如何影响情感记忆

### 相关理论

- 具身认知（Embodied Cognition）
- 现象学（Phenomenology）
- 空间叙事（Spatial Narrative）
