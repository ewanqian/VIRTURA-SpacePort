# Knowledge Network

> 中文版本：[README.md](README.md)

Knowledge Network is the core learning and practice system of VIRTURA-SpacePort, an **online evolving intellectual equipment**, providing a complete path from beginner to proficient for learners and creators in **immersive experience**, **digital content creation**, and **artificial intelligence** fields.

---

## What is Knowledge Network?

Knowledge Network is not linearly arranged by stages, but rather **8 interconnected knowledge domain categories**, allowing users to choose the field they are interested in learning.

Knowledge Network emphasizes:
- **Shortest Path**: Help beginners quickly cross gaps
- **Multiple Implementation Methods**: Each module lists 2-3 solutions
- **困境预设**: Proactively inform possible problems and solutions
- **Interdisciplinary**: Knowledge domains are interconnected
- **Digital Content Creation**: Support immersive experience, digital art creation
- **Artificial Intelligence**: AI generation, AI-assisted creation, local model deployment
