## 核心工具

### Unity XR 插件

**VR Interaction Framework**
- 插件链接：[https://assetstore.unity.com/packages/templates/systems/vr-interaction-framework-161066](https://assetstore.unity.com/packages/templates/systems/vr-interaction-framework-161066)
- 适配设备：✅ Meta Quest（完全适配），⚠️ Apple Vision Pro（兼容性较差）
- 核心功能：快速配置角色控制、场景查看，无需手动设置相机、左右手交互

### SceneForge

场景锻造 - 沉浸式网页发布框架。

[链接到 SceneForge 仓库](https://github.com/ewanqian/SceneForge)

**对应知识网络阶段**：[02-build 实现构建阶段](../knowledge-network/02-build/sceneforge.md)

### LiveForge

现场锻造 - 全链路演出工作流。

[链接到 LiveForge 仓库](https://github.com/ewanqian/LiveForge)

**对应知识网络阶段**：[02-build 实现构建阶段](../knowledge-network/02-build/liveforge.md)
