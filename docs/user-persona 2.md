## 二、人群画像

### 人群1：XR探索者 XR Explorer

**对应知识领域**：空间计算/设备

#### 人口统计

- **年龄**：22-35岁
- **职业**：科技爱好者、VR/AR开发者、独立创作者
- **教育背景**：理工科、艺术设计
- **设备**：已持有或计划购买 Apple Vision Pro / Meta Quest

#### 动机

- 对新技术好奇，想第一时间体验
- 想制作自己的VR/AR作品
- 寻找展示自己创意的平台
- 探索"空间计算"这个新领域

#### 痛点

- 不知道从哪里开始
- 官方文档太复杂，看不懂
- 不知道什么设备适合自己
- 担心技术门槛太高

#### 行为特征

- 关注科技媒体、社交媒体
- 喜欢尝试新产品
- 愿意分享自己的体验
- 会主动搜索教程

#### 使用状态

```
入门期（0-7天）
├── 目标：发布第一个VR Demo
├── 状态：兴奋、好奇、有点迷茫
├── 行为：浏览Apple Vision Pro官方文档、搜索教程
└── 障碍：不知道哪个教程适合自己

成长期（7-30天）
├── 目标：制作一个完整的空间体验
├── 状态：投入、有成就感
├── 行为：反复尝试、调试、请教社区
└── 障碍：遇到技术问题卡住

成熟期（30天+）
├── 目标：做出有特色的作品
├── 状态：追求深度
├── 行为：研究底层原理、尝试新技术
└── 障碍：缺乏灵感、不知道如何创新
```

### 人群2：3D创作者 3D Creator

**对应知识领域**：Blender实时引擎、3D重建/可视化

#### 人口统计

- **年龄**：25-40岁
- **职业**：3D设计师、视觉艺术家、游戏开发者、文物数字化工作者
- **教育背景**：艺术设计、计算机
- **技能**：有Blender/3ds Max/Maya基础

#### 动机

- 想把Blender当实时引擎用，实现快速迭代
- 探索AI生成3D的新方法
- 学习高斯点云技术
- 把现实物品数字化保存

#### 痛点

- Blender渲染太慢，影响效率
- 不知道实时渲染和离线渲染的区别
- AI生成3D效果不够好
- 高斯点云技术门槛高

#### 行为特征

- 关注Blender社区、ArtStation
- 会主动学习新技术
- 有自己的作品集
- 参与线下/线上艺术活动

#### 使用状态

```
探索期（0-14天）
├── 目标：了解实时渲染概念
├── 状态：怀疑（"实时渲染能有好效果吗？"）
├── 行为：搜索Eevee教程、对比效果
└── 障碍：担心质量不如Cycles

实践期（14-60天）
├── 目标：建立实时渲染工作流
├── 状态：逐渐认可
├── 行为：尝试Eevee→几何节点→AI生成
└── 障碍：工作流不顺畅，需要手动操作

整合期（60天+）
├── 目标：建立完整的3D生产管线
├── 状态：追求效率
├── 行为：自动化脚本、批量处理
└── 障碍：需要更多AI工具集成
```

- 缺乏系统化的学习资源

#### 行为特征

- 关注音乐节、演出、展览

```

├── 状态：好奇、期待
├── 行为：搜索"音乐可视化"教程

进阶期（7-30天）
├── 目标：实现复杂的音画互动
├── 状态：专注、投入
├── 行为：研究OSC、学习TouchDesigner
└── 障碍：同步精度不够，延迟明显

专业期（30天+）
├── 目标：用于现场演出
├── 状态：追求稳定
├── 行为：学习硬件同步、信号转换
└── 障碍：需要可靠的设备和工作流
```

## 四、各人群的"最短路径"

### XR探索者

```
第1天：Apple Vision Pro新手指南 → 30分钟发布Demo
第2-3天：了解交互方式（手势/眼动/语音）
第4-7天：做一个简单的空间体验
```

```
第1-5天：Eevee快速渲染入门 → 5分钟出图
第6-14天：几何节点基础 → 程序化生成
第15-30天：AI生成3D + 高斯点云
```

```
第1天：快速频谱分析 → 音乐驱动简单动画
第2-3天：了解OSC同步
第4-7天：TouchDesigner/Notch入门
```

```
第1-3天：Spline快速Demo
第4-7天：Three.js基础
第8-14天：Blender导出工作流
第15-30天：性能优化

```
第1-3天：了解整体工作流

第8-14天：培训团队、试运行
```

### 叙事者

```
第1-2天：空间叙事理论
第3-7天：分镜头脚本练习
第8-14天：服务设计基础
```

### 表演艺术者

```
第1-3天： rhizomatiks/teamLab案例
第4-7天：具身理论入门
