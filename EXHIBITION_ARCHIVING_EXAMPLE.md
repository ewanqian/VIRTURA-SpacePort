# 展览归档示例

> **MOVA练习课题** - 把展览当作出书，把视觉、文本、视频都编辑到一起
> **状态**: 练习/示例

---

## 什么是展览归档？

把展览当作出书！

- **视觉** - 图片、视频、沉浸式体验
- **文本** - 作品说明、艺术家介绍、策展理念
- **视频** - 现场记录、访谈、花絮
- **都编辑到一起** - 形成完整的展览档案

---

## 为什么要做展览归档？

1. **资产沉淀** - 一次创作，多场景复用
2. **虚实共生** - 把数字展览当作真实展览对待
3. **练习课题** - 作为MOVA的练习课题
4. **标准化流程** - 形成可复用的展览归档流程

---

## 展览归档内容清单

### 1. 视觉内容

| 类型 | 说明 | 格式 |
|-----|------|------|
| 作品图片 | 参展作品的高清图片 | JPG/PNG/TIFF |
| 展览现场照片 | 展览现场的记录照片 | JPG/PNG |
| 沉浸式体验截图 | 沉浸式体验的截图 | PNG |
| 展览视觉设计 | 展览的视觉设计文件 | AI/PSD/SVG |

### 2. 文本内容

| 类型 | 说明 |
|-----|------|
| 展览标题 | 展览的标题和副标题 |
| 策展理念 | 策展人的理念阐述 |
| 艺术家介绍 | 参展艺术家的介绍 |
| 作品说明 | 每个参展作品的说明 |
| 展览导览 | 展览的导览文字 |
| 新闻稿 | 展览的新闻稿 |

### 3. 视频内容

| 类型 | 说明 | 格式 |
|-----|------|------|
| 展览现场视频 | 展览现场的视频记录 | MP4/MOV |
| 艺术家访谈 | 参展艺术家的访谈视频 | MP4/MOV |
| 策展人访谈 | 策展人的访谈视频 | MP4/MOV |
| 花絮 | 展览幕后的花絮视频 | MP4/MOV |
| 沉浸式体验录像 | 沉浸式体验的录像 | MP4/MOV |
