## 📚 知识网络 Knowledge Network

知识网络是 VIRTURA-SpacePort 的核心学习与实践体系，为沉浸式体验领域的学习者和创作者提供从入门到精通的完整路径。

以"沉浸式体验"为核心，构建"入门启蒙→进阶实现→运营维护→可持续发展"的完整知识链条；以国际一流大学本科课程为标杆，兼顾理论深度与实操落地，重点降低Unity实操门槛，融入轻量化实操技巧与便捷插件用法。

## 什么是VIRTURA-SpacePort？

VIRTURA-SpacePort是VIRTURA太空港体系的**主港**和**容器**，是整个体系的孵化中心：

- **太空港（SpacePort）** - 主港、容器、孵化状态
- **太空站（SpaceStation）** - 具体的单元（MOVA美术馆、数字茶屋、演出空间等）
- **虚实共生** - 把数字空间当作真实空间对待

目前还没有哪个太空站能完全独立出来，都在太空港里孵化！

## 核心原则

1. **虚实共生原则** - 把数字空间当作真实空间对待
2. **去中心化原则** - 每个太空站相对独立，但都在太空港孵化
3. **工程化思维原则** - 系统化、可复用、可迭代
4. **长期主义原则** - 构建长期行业壁垒
5. **极简原则** - 抓主要矛盾，不要繁复
6. **资产沉淀原则** - 一次创作，多场景复用
