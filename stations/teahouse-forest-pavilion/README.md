## 技术实现

### 核心技术

- **场景采集**：iPhone LiDAR + 摄影测量
- **AI增强**：ComfyUI + Sharp 图像→高斯点云
- **实时渲染**：Blender Eevee / Supersplat
- **空间计算**：Apple Vision Pro

### 工作流

```
现实场景 → LiDAR扫描 → Mesh处理 → AI增强 → 高斯点云 → 实时渲染 → 空间体验
```
