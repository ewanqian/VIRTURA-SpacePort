# 滴流 Drop Flow 系列

> 空间叙事与沉浸体验系列

---

## 核心理念

**在Drop之后，并不必然迎来爆炸，而可能是一种持久的流动，与一场关于感知的平静觉醒。**

这是所有滴流版本万变不离其宗的核心：音乐drop之后的状态，不是激烈的峰值狂欢，而是激烈过后的空缺、下坠后的平静，所有的画面、声音、情绪，都从这一滴坠落的水出发，逐层渗透、持续流动，最终又归于寂静，完成一场从无到有、再归于无的完整闭环。

## 研究与发展历程

### 第1阶段：黑白版本（1.0原型版，2025.01-06）
通过OSApython实时控制的Blender几何节点模拟系统，纯粹的黑白灰度视觉，无任何多余色彩、具象元素与自然意象，核心呈现「从一个点触发的流动秩序」。

### 第2阶段：光投影以假乱真（2.0创作营实验版，2025.08-09）
建构屏幕空间：把屏幕边界模糊，制造裸眼3D包裹状态。通过"透明通道"的轮廓结构与屏幕内光源、雾化介质的配合，让影像以阴影与体积光的形式进入现实空间。

### 第3阶段：高斯点云数字花园（3.0展演版，2025.07）
在2.0基础上加入高斯点云技术，融入实地扫描的真实自然场景三维数据，生成可实时生长、实时变形的数字自然意象，搭配参数化立方体阵列与动态红色丝线。

### 第4阶段：空间叙事与沉浸体验（杭州双年展版，2025.10）
舞者误入信息洋流包裹的数字世界，在滴流声的张弛间，经历紧张、好奇到融入的深潜之旅。万物以蒙太奇与几何切片的形态倏忽流淌，于绚丽绽放与悄然湮灭间，演绎数字时空的感知流变。

### 第5阶段：获奖与展览（重庆展演版，2025.11）
2025「流光绘影」光影科技艺术节（LUMINOUS FLARE LIGHT & TECH ART FESTIVAL）优秀作品奖，主办单位：四川美术学院、重庆市南岸区文化和旅游发展委员会、重庆市美术家协会，活动地点：重庆市南岸区重庆开埠遗址公园，活动时间：2025.11.8-2025.11.13。

### 第6阶段：未来规划（2026规划版）
从"被动观看"到"主动探索"——观众不再是外部观察者，而是进入数字时空的参与者。实时引擎构建可漫游的数字空间，头盔内的双目立体影像，突破屏幕/投影的平面边界，进入式的沉浸体验。多人协同交互：观众可以在数字空间中自由移动，与其他观众相遇、协作，共同探索滴流世界的每个角落。
