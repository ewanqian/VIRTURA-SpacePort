# 滴流 Drop Flow 系列

空间叙事与沉浸体验系列

---

## 核心理念

**不把屏幕当作发光的平面，而把它当作光学引擎。**

通过"透明通道"的轮廓结构与屏幕内光源、雾化介质的配合，让影像以阴影与体积光的形式进入现实空间，从而制造一种裸眼3D的视觉欺骗，让画面像物体一样存在。

## 三阶段研究

### 第1部分：黑白版本（1.0原型版）
通过OSApython实时控制的Blender几何节点模拟系统

### 第2部分：光投影以假乱真（2.0创作营实验版）
建构屏幕空间：把屏幕边界模糊，制造裸眼3D包裹状态

### 第3部分：高斯点云数字花园（3.0展演版）
在2.0基础上加入高斯点云
